# Web-Based Dormitel Booking System (RM Dormitel)

An interactive, responsive web application built for transient guests, students, and administrative staff at RM Dormitel.

## Project Description
The RM Dormitel Booking System simplifies room reservations for short-term and transient stays. It features transparent pricing layouts (₱500, ₱800, ₱1500), streamlined guest data documentation fields, an inline GCash receipt attachment flow, and a centralized administrative management panel for updating room availability status trackers.
